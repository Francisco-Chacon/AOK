Sistema de Gestión Local

MAKE IT TO HAPPEN LLC — Guía Rápida

📋 Abrir el Sistema

  1. Doble clic en SistemaGestion*.exe
  2. Espere a que cargue la ventana
  3. ¡Listo para usar!

👤 Crear un Cliente

  1. Sidebar → Clientes
  2. Botón "Nuevo Cliente"
  3. Ingrese nombre, teléfono, email
  4. "Guardar"

🧾 Crear una Factura

  1. Sidebar → Facturas
  2. Botón "Nueva Factura"
  3. Seleccione cliente
  4. Ingrese descripción, monto
  5. "Guardar"

📄 Exportar a PDF

  1. Abra Vista Previa
  2. Botón "Exportar PDF"
  3. El archivo se descarga solo

💾 Hacer un Backup

  1. Sidebar → Backups
  2. Botón "Crear backup"
  3. Confirme la operación
  4. Puede descargarlo a un USB

🌐 Cambiar Idioma

Icono 🌐 en la barra superior.
Alterna entre Español e Inglés.
