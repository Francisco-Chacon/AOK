1. Introducción

El Sistema de Gestión Local es una aplicación de escritorio diseñada para administrar clientes, facturas, estimados, propuestas, hojas de ruta, contratos y respaldos. Funciona completamente sin conexión a internet (excepto el Asistente IA, que requiere conexión si se usa).

Todos los datos se almacenan de forma local en el equipo, sin necesidad de servidores externos ni suscripciones.

2. Inicio del Sistema

Requisitos

  • Windows 10 o superior
  • No requiere instalación de Node.js ni ningún otro programa

Cómo abrir el sistema

  1. Localice el archivo SistemaGestion*.exe (entregado por el desarrollador).
  2. Haga doble clic para ejecutarlo.
  3. Espere unos segundos mientras se inicia el sistema (verá la ventana principal aparecer automáticamente).
  4. El sistema está listo para usar.
Nota: La primera vez que se ejecuta, el sistema crea automáticamente la base de datos y las tablas necesarias. No requiere ninguna configuración inicial.

Cómo cerrar el sistema

  • Cierre la ventana normalmente, o presione Alt + F4.
  • El sistema guarda todo automáticamente antes de cerrarse.

  • Área central: Contenido del módulo seleccionado.

4. Clientes

Lista de clientes

Al entrar en Clientes, se muestra una lista con todos los clientes registrados. Puede buscar por nombre usando la barra de búsqueda. La lista se actualiza automáticamente al crear, editar o eliminar un cliente.

Crear un cliente

  1. Haga clic en "Nuevo Cliente".
  2. Complete los campos:
    • Nombre: Nombre del cliente (obligatorio).
    • Dirección: Dirección física.
    • Teléfono: Seleccione el código de país y escriba el número. El código de país tiene banderas para 17 países.
    • Email: Correo electrónico.
    • Estado: Activo, Pendiente o Inactivo.
  3. Haga clic en "Guardar".
Nota: Los campos de teléfono y email tienen validación inline: si el formato no es correcto, se mostrará un mensaje en rojo, pero no impide guardar.

Editar un cliente

  1. Haga clic en el botón editar (icono de lápiz) en la tarjeta del cliente.
  2. Modifique los campos necesarios.
  3. Haga clic en "Guardar".

Eliminar un cliente

  1. Haga clic en el botón eliminar (icono de papelera).
  2. Confirme la eliminación en el modal que aparece.

Ver detalle

Haga clic en la tarjeta del cliente para ver un modal con la información completa, incluyendo un avatar y el badge de estado.

5. Facturas

Lista de facturas

Muestra todas las facturas creadas. Puede filtrar por estado (Pendiente, Pagado, Anulado) usando el selector en la parte superior. También puede buscar por cliente.

Crear una factura

  1. Haga clic en "Nueva Factura".
  2. Complete los campos:
    • INVOICE #: Número de factura (personalizable, por defecto "INV-{id}").
    • Cliente: Seleccione de la lista desplegable con búsqueda. Al seleccionar, se cargan automáticamente dirección, email y teléfono.
    • Descripción: Detalle del servicio o producto.
    • Monto: Valor de la factura.
    • Fecha: Fecha de emisión.
    • Estado: Pendiente, Pagado o Anulado.
  3. Haga clic en "Guardar".

Editar / Eliminar

Use los botones de acción en cada fila de la tabla (editar y eliminar).

Vista previa e impresión

  1. Haga clic en el botón "Vista Previa" en la fila de la factura.
  2. Se abrirá un panel con el diseño de la factura estilo profesional.
  3. Desde la vista previa puede hacer clic en "Imprimir" o "Exportar PDF".

6. Hojas de Ruta

Lista de hojas de ruta

Muestra todas las hojas de ruta registradas. Puede buscar por fecha y ordenar los resultados.

Crear una hoja de ruta

  1. Haga clic en "Nueva Hoja de Ruta".
  2. Complete los datos generales:
    • Fecha: Día de la ruta.
    • Conductor: Nombre del conductor.
    • Camión: Identificación del vehículo.
  3. Agregue clientes a la ruta:
    • Seleccione un cliente de la lista desplegable.
    • Ingrese hora de entrada y hora de salida.
    • Agregue una descripción de la visita.
    • Haga clic en "Agregar" para incluir al cliente en la tabla de la ruta.
  4. Puede agregar múltiples clientes a una misma ruta. Cada cliente aparece en una tabla con sus horarios.
  5. Para eliminar un cliente de la ruta, use el botón de eliminar en la fila correspondiente.
  6. Haga clic en "Guardar".

Editar / Eliminar

Use los botones de acción en cada fila de la tabla de hojas de ruta.

Vista previa e impresión

Desde la vista previa puede ver el documento completo con todos los clientes y horarios, listo para Imprimir o Exportar PDF.

7. Estimados (Presupuestos)

Lista de estimados

Muestra todos los presupuestos creados. Puede filtrar por estado (Borrador, Enviado, Aceptado, Rechazado) y buscar por cliente.

Crear un estimado

  1. Haga clic en "Nuevo Estimado".
  2. Complete los datos generales:
    • Cliente: Seleccione de la lista desplegable.
    • Dirección de trabajo: Dirección donde se realizará el trabajo.
    • Fecha: Fecha del presupuesto.
    • Moneda: USD, EUR, MXN, PAB o COP.
    • Tasa de impuesto: Seleccione 7%, 15% o 25%.
    • Estado: Borrador, Enviado, Aceptado o Rechazado.
  3. Agregue materiales o servicios (ítems):
    1. Descripción: Detalle del ítem (ej: "Mano de obra", "Material eléctrico").
    2. Cantidad: Número de unidades (por defecto 1).
    3. Precio unitario: Valor por unidad.
    4. Haga clic en "Agregar" para incluir el ítem en la tabla.
  4. El sistema calcula automáticamente:
    • Subtotal: Suma de (cantidad × precio unitario) de todos los ítems.
    • Impuesto: Subtotal × tasa de impuesto seleccionada.
    • Total: Subtotal + impuesto.
  5. Haga clic en "Guardar".

Editar / Eliminar

Use los botones de acción en cada fila de la tabla de estimados.

Vista previa e impresión

La vista previa se muestra en un panel dividido: a la izquierda los datos del estimado, a la derecha el documento renderizado. Puede Imprimir o Exportar PDF.

8. Propuestas

Lista de propuestas

Muestra todos los estimados que pueden convertirse en propuestas formales. Puede filtrar por estado y buscar por cliente.

Crear una propuesta

  1. Haga clic en "Nueva Propuesta".
  2. Complete los campos: cliente, dirección de trabajo, fecha, moneda, descripción, monto y estado.
  3. Haga clic en "Guardar".
  4. Al guardar, la propuesta se muestra automáticamente en el panel de vista previa del lado derecho.

Vista previa

La propuesta se renderiza como un documento formal imprimible con:

  • Logo de la empresa
  • Datos del cliente (nombre, dirección, email, teléfono)
  • Dirección de trabajo
  • Descripción detallada del trabajo
  • Monto y moneda
  • Diseño profesional listo para imprimir

Exportar PDF

Desde la vista previa, haga clic en "Exportar PDF". El archivo se guardará con el nombre traducido según el idioma activo.

9. Contratos

Lista de contratos

Muestra todos los contratos registrados. Puede buscar por cliente usando la barra de búsqueda. Cada fila muestra el cliente, fechas y monto.

Crear un contrato

  1. Haga clic en "Nuevo Contrato".
  2. Complete los campos:
    • Cliente: Seleccione de la lista desplegable con búsqueda.
    • Fecha de inicio: Fecha en que comienza el contrato.
    • Fecha de fin: Fecha de finalización del contrato.
    • Descripción: Detalle del contrato (alcance, términos, etc.).
    • Monto: Valor total del contrato.
  3. Haga clic en "Guardar".
Validación: Si la fecha de fin es anterior a la fecha de inicio, se muestra un error en rojo en el campo de fecha. Corrija las fechas antes de guardar.

Vista previa

La vista previa del contrato incluye:

  • Tabla con datos del cliente (nombre, dirección, email, teléfono)
  • Fechas de inicio y fin
  • Descripción del contrato en un recuadro
  • Monto del contrato
  • Líneas de firma: Firma del Cliente y Firma del Contratista

Imprimir / Exportar PDF

Use los botones correspondientes desde la vista previa. El nombre del archivo incluye la traducción del módulo según el idioma activo.

10. Backups (Respaldos)

Importante: Hacer backups regularmente es la mejor forma de proteger sus datos. Se recomienda crear un backup al menos una vez por semana y guardar una copia en un USB o en la nube.

Crear un backup

  1. Vaya a la sección Backups en la barra lateral.
  2. Haga clic en "Crear backup".
  3. Aparecerá un mensaje de confirmación. Haga clic en "Sí, crear backup".
  4. Espere unos segundos mientras se crea el respaldo.
  5. El backup aparecerá en la lista con la fecha y hora de creación.

Descargar un backup

Haga clic en "Descargar" junto al backup que desea guardar. El archivo .db se descargará en su computadora.

Subir un backup

  1. En la sección "Subir backup desde archivo", haga clic en el botón de selección de archivo.
  2. Seleccione un archivo .db desde su computadora (USB, disco duro, etc.).
  3. Haga clic en "Subir backup".
  4. El archivo aparecerá en la lista de backups disponible para restaurar.

Restaurar un backup

  1. Haga clic en "Restaurar" junto al backup que desea recuperar.
  2. Aparecerá un mensaje de advertencia: "Esta acción no se puede deshacer".
  3. Confirme la operación.
  4. El sistema reemplazará todos los datos actuales con los del backup.
  5. Reinicie el sistema para asegurar que todos los cambios se apliquen correctamente.

Eliminar un backup

  1. Haga clic en el botón de eliminar (icono de papelera) junto al backup.
  2. Confirme la eliminación en el modal que aparece.
  3. El archivo se borrará permanentemente.

11. Asistente IA

El Asistente IA es un chat integrado que puede ayudar a usar el sistema. Tiene dos modos de funcionamiento.

Cómo abrirlo

Haga clic en el botón flotante (icono de robot 🤖) en la esquina inferior derecha de la pantalla. Se abrirá un panel de chat.

Consultas de datos (sin internet)

El asistente puede responder preguntas sobre los datos del sistema sin conexión a internet. Ejemplos:

  • "¿Cuántos clientes tengo?"
  • "Resumen general del sistema"
  • "¿Qué facturas están pendientes?"
  • "Mostrar contratos activos"
  • "Actividad reciente"
  • "Buscar [nombre del cliente]"
  • "¿Cuántos estimados tengo?"

Conversación con IA (requiere internet + API key)

Para hacer preguntas como "¿Cómo creo una factura?" o "¿Qué es una hoja de ruta?", se necesita:

  1. Tener conexión a internet.
  2. Configurar una API key de OpenRouter (ver Configuración).
Nota: Si no hay API key configurada, el asistente mostrará un mensaje amigable indicando qué consultas de datos están disponibles y apuntando al icono de engranaje ⚙️ para configurarla.

12. Configuración (API Key)

La API key permite que el Asistente IA responda preguntas sobre el uso del sistema usando inteligencia artificial.

Cómo configurarla

  1. Haga clic en el icono de engranaje ⚙️ en la barra superior.
  2. Se abrirá un modal con un campo para ingresar la API key.
  3. Escriba su API key de OpenRouter (comienza con sk-or-v1-).
  4. Haga clic en "Guardar".
  5. Aparecerá un checkmark verde ✅ indicando que la key está configurada correctamente.
Seguridad: La API key se almacena cifrada en la base de datos local usando AES-256-GCM. Nunca se muestra en la interfaz: si ya hay una key configurada, el campo aparece vacío por seguridad.

Cómo eliminar la API key

  1. Abra la configuración (icono de engranaje ⚙️).
  2. Haga clic en "Eliminar".
  3. La key se borrará permanentemente de la base de datos.

15. Solución de Problemas

El sistema no se abre

  • Asegúrese de tener Windows 10 o superior.
  • Ejecute el .exe como Administrador (clic derecho → "Ejecutar como administrador").
  • Revise que su antivirus no haya bloqueado el programa. Si es necesario, agregue el .exe a las exclusiones del antivirus.

No aparecen los datos después de restaurar un backup

  • Reinicie el sistema después de restaurar.
  • Si el problema persiste, contacte a Soporte Técnico.

El Asistente IA no responde

  • Sin internet: Use solo consultas de datos (pregunte por clientes, facturas, contratos, etc.).
  • Con internet pero no responde: Verifique que la API key esté configurada en Configuración (⚙️).
  • La API key es incorrecta: Elimine la key actual en Configuración e ingrese una nueva.

Error al exportar PDF

  • Cierre la ventana de vista previa y vuelva a abrirla.
  • Si el error persiste, reinicie el sistema e intente nuevamente.
  • Verifique que tenga espacio en disco para guardar el archivo.

Los textos aparecen en otro idioma

  • Use el botón de idioma 🌐 en la barra superior para alternar entre Español e Inglés.

No encuentro un cliente en la lista

  • Use la barra de búsqueda en la parte superior de la lista de clientes.
  • Revise los filtros de estado si están activos.
  • La lista muestra 20 clientes por página. Use la paginación en la parte inferior para ver más páginas.

Olvidé mi API key de OpenRouter

  • No se puede recuperar la key desde el sistema por razones de seguridad.
  • Elimine la key actual en Configuración (⚙️ → "Eliminar") e ingrese una nueva.
  • Si no tiene una key, regístrese en openrouter.ai para obtener una.
